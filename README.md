# 📊 E-commerce + Services System (SQL)

This project consists of the complete modeling of a relational database for an **e-commerce system with technical service support**, developed using **MySQL**.

The goal is to simulate a real-world application environment, including user registration, product catalog, orders, payments, inventory, and service orders.

---

## 🚀 Modeled Features

- User and address management
- Product and service catalog
- Inventory control
- Order system with multiple items
- Payment processing
- Service order management
- Relationships with referential integrity (FKs)

---

## 🧠 Project Purpose

Simulate a real database environment by applying relational modeling concepts, referential integrity, and SQL queries in an e-commerce scenario with services.

This project covers:

- Relational modeling
- Table creation with constraints
- Use of foreign keys (FK)
- Writing SQL queries (basic and advanced)
- SQL script organization
- Realistic data simulation (seeds)

---

## 🛠️ Technologies Used

- MySQL
- SQL (DDL + DML)
